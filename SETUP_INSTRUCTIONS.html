Database Update Required

The new advanced SQL features (triggers, functions, ENUM types) have been added to the code, but your database needs to be updated with the new schema.

Setup Steps

1

Visit the Database Initialization Page

Click the button below to go to the initialization page:

Go to init_db.php
2

Reset the Database

On the initialization page, click the "Reset Database" link. This will drop all old tables and recreate them with the new schema including:

  • ENUM data types (customer_tier, stock_status, order status, delivery_status)
  • New columns (loyalty_points, discount_applied)
  • 4 Database Triggers (auto-calculate subtotals, update totals, loyalty points, price validation)
  • 2 User-Defined Functions (calculate_discount, get_customer_tier)
3

View the Advanced Features Demo

After resetting, visit the demo page to see all the new features in action:

View Advanced Features Demo
